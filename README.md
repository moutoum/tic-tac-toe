# Tic Tac Toe

This project is a simple Tic Tac Toe game.
It was created to learn the [Rust Programming Language](https://www.rust-lang.org/).

# Use

## Install the `rust` dependencies
```sh
curl https://sh.rustup.rs -sSf | sh
```

## Run project
```sh
cargo run --release
```

## Run tests
```sh
cargo test
```
